# data-pipeline
A docker environment with tippecanoe, mapshaper, R, csvkit, python, node, etc. ready to go


## Common Tasks

### Extracting Data

- Downloading from S3
- Downloading from API endpoint

### Transforming Data

- JSON
- Static CSV
- GeoJSON
- Vector Tiles

### Loading Data

- Static file to Database
- Uploading to S3
- Deploying vector tiles to Mapbox
