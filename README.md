# data-pipeline
A docker environment with tippecanoe, mapshaper, R, csvkit, python, node, etc. ready to go


## Common Tasks

### Extracting Data

- Downloading from S3
- Downloading from API endpoint


### Transforming Data

- Join / Merge
- Slice
- Group
- Map Tiles

### Loading Data

- Deploying to Database
- Uploading to S3
- Deploying to Mapbox
